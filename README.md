### Light ###

Shadowsocks 多用户多区域自助售卖面板，基于 Laravel 与 Vue 编写。

Light 是一个很快的应用，在启用 Gzip 和 Production 模式后，首屏加载 200k，在服务器有 4M 宽带情况下从访问到渲染完成不超过 1 秒。

特性：

 - 单页面应用
 - 用户自助
     - 注册
     - 购买
     - 开通
     - 续费
     - 编辑信息
     - 重置密码
 - 消费记录
 - 显示二维码
 - 流量计费 / 时间计费
 - 套餐激活码
 - 响应式界面
 - 支持多服务器区域
 - 完整后台功能支持
 - 一键创建用户，开通帐号
